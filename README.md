## Run 

Untuk menjalankan aplikasi ini, bisa dilakukan dengan menjalankan beberapa command di terminal.

1. Pertama run command dibawah untuk compile code nya
```bash
make
```

2. lalu run binary codenya
```bash
build/spotify
```
